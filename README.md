# Tic-Tac-Toe CLI

A text-based two-player Tic-Tac-Toe game that runs in the terminal, built in Python.

## Features
- Two player gameplay with alternating turns
- Input validation and error handling
- Flexible input format (1-A, 1A, and 1 A all work)
- Win detection across all rows, columns, and diagonals
- Draw detection
- Play again functionality

## Requirements
- Python 3
- pyfiglet

## Installation
pip install pyfiglet

## How to Run
python main.py

## How to Play
- Players take turns entering a cell in Row-Column format
- Rows: 1, 2, 3 | Columns: A, B, C
- Valid input formats: 1-A, 1A, or 1 A
- First player to get 3 in a row wins!
